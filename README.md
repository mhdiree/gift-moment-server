## 📝 프로젝트 개요
사용자가 자신의 생일을 친구들과 쉽게 공유하고 선물 펀딩과 편지를 아카이빙할 수 있는 서비스
<br>
## 🎯 개발 목표
사용자가 자신의 생일을 친구들과 쉽게 공유하기<br>
원하는 선물을 등록하고, 여러 사람이 펀딩하여 선물하기<br>
생일 편지와 선물을 손쉽게 관리하고 아카이빙<br>
